**Flux** is a **web application** for personal finance management, allowing users to record **income and expenses**, organize transactions by **category**, and track their **monthly balance** in a simple and efficient way.

-  Income recording  
-  Expense recording  
-  Category-based classification  
-  Monthly balance overview  
-  Transaction listing  
-  Transaction filtering
